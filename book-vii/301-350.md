# 集合论习题集 第7卷：序数理论 (301-350)

## 主题：序数定义、良序集、超限归纳、序数运算、初始序数

### 第301题
**题目描述：**
证明：每个良序集都是全序集。

**解答过程：**
设 $(W, <)$ 是一个良序集。

良序的定义要求：
1. $<$ 是 $W$ 上的全序关系
2. $W$ 的每个非空子集都有最小元

根据定义，良序集首先必须是全序集。这是因为良序要求任意两个元素可比，这是全序的核心性质。

更详细的证明：
- 对于任意 $a, b \in W$，考虑子集 $\{a, b\}$
- 由良序性，$\{a, b\}$ 有最小元
- 若最小元是 $a$，则 $a \leq b$；若最小元是 $b$，则 $b \leq a$
- 因此任意两元素可比，$<$ 是全序

**答案：**
良序集必为全序集，这是良序定义的直接推论。

### 第302题
**题目描述：**
证明：有限全序集必是良序集。

**解答过程：**
设 $(F, <)$ 是有限全序集，$|F| = n < \omega$。

需证：$F$ 的每个非空子集都有最小元。

用反证法：
- 假设存在非空子集 $S \subseteq F$ 没有最小元
- 任取 $s_1 \in S$，由于 $s_1$ 不是最小元，存在 $s_2 \in S$ 使 $s_2 < s_1$
- 同理，存在 $s_3 \in S$ 使 $s_3 < s_2 < s_1$
- 继续下去，得到无限严格递减链：$\cdots < s_3 < s_2 < s_1$
- 这与 $F$ 有限矛盾

或者直接证明：
有限全序集同构于某个初始段 $\{0, 1, \ldots, n-1\}$，而后者显然是良序的。

**答案：**
有限全序集必为良序集。

### 第303题
**题目描述：**
证明：$(\mathbb{N}, <)$ 是良序集，其中 $<$ 是通常的小于关系。

**解答过程：**
需证：$\mathbb{N}$ 的每个非空子集都有最小元。

设 $S \subseteq \mathbb{N}$，$S \neq \emptyset$。

由自然数的归纳原理：
1. 若 $0 \in S$，则 $0$ 是最小元
2. 若 $0 \notin S$，考虑集合 $T = \{n \in \mathbb{N} : n < s, \forall s \in S\}$
3. $T$ 非空（包含0），且有上界
4. 由自然数的良序性（或归纳原理），$T$ 有最大元 $m$
5. 则 $m + 1 \in S$（否则 $m+1 \in T$，与 $m$ 最大矛盾）
6. 且 $m+1$ 是 $S$ 的最小元

**答案：**
$(\mathbb{N}, <)$ 是良序集，这是自然数的基本性质。

### 第304题
**题目描述：**
证明：$(\mathbb{Z}, <)$ 不是良序集。

**解答过程：**
良序要求每个非空子集有最小元。

考虑子集 $S = \{\ldots, -2, -1, 0\} \subseteq \mathbb{Z}$（所有非正整数）。

假设 $S$ 有最小元 $m$。
- 则 $m \in S$，即 $m \leq 0$
- 但 $m - 1 \in S$ 且 $m - 1 < m$
- 这与 $m$ 是最小元矛盾

或者考虑 $S = \mathbb{Z}$ 本身：
- 若 $m$ 是 $\mathbb{Z}$ 的最小元，则 $m - 1 < m$ 且 $m - 1 \in \mathbb{Z}$
- 矛盾

**答案：**
$(\mathbb{Z}, <)$ 不是良序集，因为整数集没有最小元。

### 第305题
**题目描述：**
证明：$(\mathbb{Q}^+, <)$ 不是良序集，其中 $\mathbb{Q}^+$ 是正有理数集。

**解答过程：**
考虑子集 $S = \{\frac{1}{n} : n \in \mathbb{N}, n \geq 1\} \subseteq \mathbb{Q}^+$。

假设 $S$ 有最小元 $\frac{1}{n_0}$。
- 但 $\frac{1}{n_0 + 1} \in S$ 且 $\frac{1}{n_0 + 1} < \frac{1}{n_0}$
- 这与 $\frac{1}{n_0}$ 是最小元矛盾

因此 $S$ 没有最小元，$\mathbb{Q}^+$ 不是良序集。

**答案：**
$(\mathbb{Q}^+, <)$ 不是良序集。

### 第306题
**题目描述：**
设 $(W_1, <_1)$ 和 $(W_2, <_2)$ 都是良序集。证明它们的字典序积 $W_1 \times W_2$ 也是良序集。

**解答过程：**
字典序定义为：$(a, b) < (c, d)$ 当且仅当 $a <_1 c$ 或 ($a = c$ 且 $b <_2 d$)。

设 $S \subseteq W_1 \times W_2$，$S \neq \emptyset$。

**构造最小元：**
1. 令 $A = \{a \in W_1 : \exists b \in W_2, (a, b) \in S\}$
2. 由 $W_1$ 良序，$A$ 有最小元 $a_0$
3. 令 $B = \{b \in W_2 : (a_0, b) \in S\}$
4. 由 $W_2$ 良序，$B$ 有最小元 $b_0$
5. 则 $(a_0, b_0)$ 是 $S$ 的最小元

验证：对任意 $(a, b) \in S$，
- 若 $a >_1 a_0$，则 $(a, b) > (a_0, b_0)$
- 若 $a = a_0$，则 $b \in B$，故 $b \geq_2 b_0$，从而 $(a, b) \geq (a_0, b_0)$

**答案：**
良序集的字典序积仍是良序集。

### 第307题
**题目描述：**
证明：良序集的任意非空子集仍是良序集。

**解答过程：**
设 $(W, <)$ 是良序集，$S \subseteq W$，$S \neq \emptyset$。

需证：$(S, <)$ 是良序集。

1. **全序性：** 由于 $<$ 是 $W$ 上的全序，限制在 $S$ 上仍是全序

2. **良序性：** 设 $T \subseteq S$，$T \neq \emptyset$
   - 则 $T \subseteq W$ 且 $T \neq \emptyset$
   - 由 $W$ 良序，$T$ 在 $W$ 中有最小元 $t_0$
   - $t_0 \in T \subseteq S$，且对所有 $t \in T$，$t_0 \leq t$
   - 故 $t_0$ 也是 $T$ 在 $S$ 中的最小元

**答案：**
良序集的任意非空子集仍是良序集。

### 第308题
**题目描述：**
证明：良序集不存在无限严格递减序列。

**解答过程：**
设 $(W, <)$ 是良序集。

假设存在无限严格递减序列 $a_0 > a_1 > a_2 > \cdots$。

考虑集合 $S = \{a_n : n \in \mathbb{N}\}$。
- $S \subseteq W$ 且 $S \neq \emptyset$
- 由良序性，$S$ 有最小元 $a_k$
- 但 $a_{k+1} \in S$ 且 $a_{k+1} < a_k$
- 这与 $a_k$ 是最小元矛盾

因此良序集不存在无限严格递减序列。

**答案：**
良序集不存在无限严格递减序列，这是良序性的等价刻画。

### 第309题
**题目描述：**
证明：若全序集不存在无限严格递减序列，则它是良序集。

**解答过程：**
设 $(W, <)$ 是全序集，且不存在无限严格递减序列。

假设 $W$ 不是良序集，则存在非空子集 $S \subseteq W$ 没有最小元。

任取 $s_0 \in S$（因 $S \neq \emptyset$）。
- 由于 $s_0$ 不是最小元，存在 $s_1 \in S$ 使 $s_1 < s_0$
- 由于 $s_1$ 不是最小元，存在 $s_2 \in S$ 使 $s_2 < s_1$
- 继续下去，得到无限严格递减序列 $s_0 > s_1 > s_2 > \cdots$
- 这与假设矛盾

因此 $W$ 是良序集。

**答案：**
全序集是良序集当且仅当它不存在无限严格递减序列。

### 第310题
**题目描述：**
定义序数 $\omega$ 为自然数集 $\mathbb{N}$ 的序型。证明：$\omega$ 是极限序数。

**解答过程：**
序数 $\alpha$ 是极限序数如果它既不是零也不是后继序数，即不存在序数 $\beta$ 使 $\alpha = \beta + 1$。

**证明 $\omega$ 是极限序数：**
- $\omega \neq 0$（显然）
- 假设 $\omega = \beta + 1$ 对某个序数 $\beta$
- 则 $\beta < \omega$，即 $\beta$ 是某个自然数 $n$
- 但 $n + 1 = \{0, 1, \ldots, n\}$ 仍是有限序数
- 而 $\omega = \{0, 1, 2, \ldots\}$ 是所有自然数的集合
- $\omega \neq n + 1$ 对任何有限 $n$

因此 $\omega$ 不是后继序数，它是极限序数。

**答案：**
$\omega$ 是极限序数，它是第一个无限序数。

### 第311题
**题目描述：**
证明：序数 $\omega + 1$ 与 $\omega$ 不同，且 $\omega < \omega + 1$。

**解答过程：**
序数加法的定义：$\alpha + \beta$ 是将 $\alpha$ 的序型后接 $\beta$ 的序型。

具体地：
- $\omega = \{0, 1, 2, \ldots\}$
- $\omega + 1 = \omega \cup \{\omega\} = \{0, 1, 2, \ldots, \omega\}$

**证明 $\omega \neq \omega + 1$：**
- $\omega + 1$ 有最大元 $\omega$
- $\omega$ 没有最大元（对任意 $n \in \omega$，存在 $n+1 > n$）
- 因此两者序型不同

**证明 $\omega < \omega + 1$：**
- $\omega$ 是 $\omega + 1$ 的真初始段
- 即 $\omega \subsetneq \omega + 1$ 且保持序关系
- 因此 $\omega < \omega + 1$

**答案：**
$\omega < \omega + 1$，且 $\omega + 1$ 有最大元而 $\omega$ 没有。

### 第312题
**题目描述：**
证明：$1 + \omega = \omega$，但 $\omega + 1 \neq \omega$。这说明序数加法不满足交换律。

**解答过程：**
**证明 $1 + \omega = \omega$：**
- $1 + \omega$ 表示将一个元素放在 $\omega$ 之前
- 构造序同构：$f: 1 + \omega \to \omega$
- $f(0) = 0$（$1$ 中的元素）
- $f(1 + n) = n + 1$ 对 $n \in \omega$
- 这是双射且保序
- 因此 $1 + \omega \cong \omega$，即 $1 + \omega = \omega$

**证明 $\omega + 1 \neq \omega$：**
- 已在第311题证明

**结论：**
$1 + \omega = \omega \neq \omega + 1$，序数加法不满足交换律。

**答案：**
$1 + \omega = \omega$ 而 $\omega + 1 \neq \omega$，序数加法不交换。

### 第313题
**题目描述：**
计算 $(\omega + 1) + \omega$ 并化简。

**解答过程：**
序数加法结合律成立：$(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)$。

计算 $(\omega + 1) + \omega$：
- 这是将 $\omega$ 接在 $(\omega + 1)$ 之后
- 即序列：$0, 1, 2, \ldots, \omega, \omega + 1, \omega + 2, \ldots$
- 这等价于 $\omega + (1 + \omega) = \omega + \omega = \omega \cdot 2$

或者直接观察：
- 在 $(\omega + 1)$ 的最后一个元素 $\omega$ 之后
- 开始一个新的 $\omega$ 序列
- 总序型为 $\omega + \omega = \omega \cdot 2$

**答案：**
$(\omega + 1) + \omega = \omega \cdot 2$

### 第314题
**题目描述：**
证明：对于任意有限序数 $n$，$n + \omega = \omega$。

**解答过程：**
有限序数 $n = \{0, 1, \ldots, n-1\}$。

**证明 $n + \omega = \omega$：**
- $n + \omega$ 表示将 $n$ 个元素放在 $\omega$ 之前
- 构造序同构 $f: n + \omega \to \omega$：
  - 对 $k < n$，$f(k) = k$
  - 对 $m \in \omega$，$f(n + m) = n + m$
- 这是双射且保序

直观理解：
- 在无限序列前添加有限个元素
- 不改变整体的序型
- 因为 $\omega$ 是极限序数，前面有限部分不影响"极限"性质

**答案：**
对任意有限序数 $n$，$n + \omega = \omega$。

### 第315题
**题目描述：**
计算 $\omega \cdot 2$ 并描述其序型。

**解答过程：**
序数乘法定义：$\alpha \cdot \beta$ 是 $\beta$ 个 $\alpha$ 的序型和。

**计算 $\omega \cdot 2$：**
- $\omega \cdot 2 = \omega + \omega$
- 这是两个 $\omega$ 序列的连接
- 序型为：$0, 1, 2, \ldots, \omega, \omega + 1, \omega + 2, \ldots$

**性质：**
- $\omega \cdot 2$ 是极限序数（没有最大元）
- $\omega \cdot 2 > \omega$
- $\omega \cdot 2$ 不是可数序数中最小的无限序数（$\omega$ 才是）

**与 $2 \cdot \omega$ 比较：**
- $2 \cdot \omega = \omega$（第316题）
- 因此序数乘法也不交换

**答案：**
$\omega \cdot 2 = \omega + \omega$，是两个自然数序列的连接。

### 第316题
**题目描述：**
证明：$2 \cdot \omega = \omega$，但 $\omega \cdot 2 \neq \omega$。

**解答过程：**
**证明 $2 \cdot \omega = \omega$：**
- $2 \cdot \omega$ 表示 $\omega$ 个 2 的序型和
- 即 $(0,0), (0,1), (1,0), (1,1), (2,0), (2,1), \ldots$
- 这可以映射到 $\omega$：$f(n, i) = 2n + i$
- 这是序同构

或者：
- $2 \cdot \omega = \sup\{2 \cdot n : n < \omega\} = \sup\{0, 2, 4, \ldots\} = \omega$

**证明 $\omega \cdot 2 \neq \omega$：**
- $\omega \cdot 2 = \omega + \omega$
- 这是两个完整的 $\omega$ 序列
- 第一个序列的所有元素都小于第二个序列的任何元素
- 这种"间隙"结构在 $\omega$ 中不存在

**答案：**
$2 \cdot \omega = \omega \neq \omega \cdot 2$，序数乘法不交换。

### 第318题
**题目描述：**
证明：对于任意序数 $\alpha$，$\alpha + 0 = \alpha$。

**解答过程：**
序数加法的递归定义：
- $\alpha + 0 = \alpha$
- $\alpha + (\beta + 1) = (\alpha + \beta) + 1$（后继）
- $\alpha + \lambda = \sup\{\alpha + \beta : \beta < \lambda\}$（极限序数）

根据定义，$\alpha + 0 = \alpha$ 是基本公理。

**直观理解：**
- 在 $\alpha$ 后接0个元素，就是 $\alpha$ 本身
- 这是加法单位元的性质

**答案：**
$\alpha + 0 = \alpha$，0是序数加法的右单位元。

### 第319题
**题目描述：**
证明：对于任意序数 $\alpha$，$0 + \alpha = \alpha$。

**解答过程：**
用超限归纳法证明。

**基例：** $0 + 0 = 0$（由定义）

**后继步：** 假设 $0 + \alpha = \alpha$，证 $0 + (\alpha + 1) = \alpha + 1$
- $0 + (\alpha + 1) = (0 + \alpha) + 1 = \alpha + 1$

**极限步：** 设 $\lambda$ 是极限序数，假设对所有 $\beta < \lambda$，$0 + \beta = \beta$
- $0 + \lambda = \sup\{0 + \beta : \beta < \lambda\} = \sup\{\beta : \beta < \lambda\} = \lambda$

因此对所有序数 $\alpha$，$0 + \alpha = \alpha$。

**答案：**
$0 + \alpha = \alpha$，0也是序数加法的左单位元。

### 第322题
**题目描述：**
陈述并证明超限归纳原理。

**解答过程：**
**超限归纳原理：**
设 $P(\alpha)$ 是关于序数 $\alpha$ 的命题。若对任意序数 $\alpha$，
$$(\forall \beta < \alpha, P(\beta)) \Rightarrow P(\alpha)$$
则对所有序数 $\alpha$，$P(\alpha)$ 成立。

**证明：**
假设结论不成立，即存在序数使 $P$ 不成立。
- 令 $S = \{\alpha : P(\alpha) \text{ 不成立}\}$
- $S \neq \emptyset$
- 由序数的良序性，$S$ 有最小元 $\alpha_0$
- 则对所有 $\beta < \alpha_0$，$P(\beta)$ 成立
- 由归纳假设，$P(\alpha_0)$ 成立
- 这与 $\alpha_0 \in S$ 矛盾

因此对所有序数，$P(\alpha)$ 成立。

**答案：**
超限归纳原理：若命题对所有小于$\alpha$的序数成立则对$\alpha$成立，则该命题对所有序数成立。

### 第323题
**题目描述：**
用超限归纳法证明：对于所有序数 $\alpha$，$\alpha \leq \omega \cdot \alpha$。

**解答过程：**
设 $P(\alpha)$：$\alpha \leq \omega \cdot \alpha$。

**基例：** $\alpha = 0$
- $0 \leq \omega \cdot 0 = 0$，成立

**后继步：** 假设 $\alpha \leq \omega \cdot \alpha$，证 $\alpha + 1 \leq \omega \cdot (\alpha + 1)$
- $\omega \cdot (\alpha + 1) = \omega \cdot \alpha + \omega$
- 由归纳假设，$\alpha \leq \omega \cdot \alpha$
- 故 $\alpha + 1 \leq \omega \cdot \alpha + 1 \leq \omega \cdot \alpha + \omega = \omega \cdot (\alpha + 1)$

**极限步：** 设 $\lambda$ 是极限序数，假设对所有 $\beta < \lambda$，$\beta \leq \omega \cdot \beta$
- $\omega \cdot \lambda = \sup\{\omega \cdot \beta : \beta < \lambda\}$
- 对任意 $\beta < \lambda$，$\beta \leq \omega \cdot \beta \leq \omega \cdot \lambda$
- 故 $\lambda = \sup\{\beta : \beta < \lambda\} \leq \omega \cdot \lambda$

由超限归纳，对所有序数 $\alpha$，$\alpha \leq \omega \cdot \alpha$。

**答案：**
对所有序数 $\alpha$，$\alpha \leq \omega \cdot \alpha$。

### 第324题
**题目描述：**
证明：若 $\alpha < \beta$，则对任意序数 $\gamma$，$\alpha + \gamma \leq \beta + \gamma$。

**解答过程：**
固定 $\alpha < \beta$，对 $\gamma$ 用超限归纳。

**基例：** $\gamma = 0$
- $\alpha + 0 = \alpha < \beta = \beta + 0$
- 故 $\alpha + 0 \leq \beta + 0$

**后继步：** 假设 $\alpha + \gamma \leq \beta + \gamma$
- $\alpha + (\gamma + 1) = (\alpha + \gamma) + 1$
- $\beta + (\gamma + 1) = (\beta + \gamma) + 1$
- 由归纳假设和序数后继的保序性，$\alpha + (\gamma + 1) \leq \beta + (\gamma + 1)$

**极限步：** 设 $\lambda$ 是极限序数
- $\alpha + \lambda = \sup\{\alpha + \gamma : \gamma < \lambda\}$
- $\beta + \lambda = \sup\{\beta + \gamma : \gamma < \lambda\}$
- 由归纳假设，对所有 $\gamma < \lambda$，$\alpha + \gamma \leq \beta + \gamma$
- 故 $\sup\{\alpha + \gamma\} \leq \sup\{\beta + \gamma\}$

**答案：**
若 $\alpha < \beta$，则对任意 $\gamma$，$\alpha + \gamma \leq \beta + \gamma$。

### 第325题
**题目描述：**
证明：对于任意序数 $\alpha$，存在极限序数 $\lambda$ 使得 $\alpha < \lambda$。

**解答过程：**
**构造方法：**

### 第337题
**题目描述：**
陈述并证明良序集的比较定理。

**解答过程：**
**比较定理：** 对于任意两个良序集 $(A, <_A)$ 和 $(B, <_B)$，下列恰有一个成立：
1. $A$ 序同构于 $B$ 的某个前段
2. $B$ 序同构于 $A$ 的某个前段
且这两种情况下同构是唯一的。

**证明概要：**
定义关系 $f = \{(a, b) \in A \times B : A_a \cong B_b\}$

**证明 $f$ 是函数：**
- 若 $A_a \cong B_b$ 且 $A_a \cong B_{b'}$
- 则 $B_b \cong B_{b'}$
- 由良序集前段唯一性，$b = b'$

**证明 $f$ 是单射：** 类似

**证明 $f$ 保序：** 由定义直接验证

**定义域和值域：**
- 若 $\text{dom}(f) = A$ 且 $\text{ran}(f) = B$，则 $A \cong B$
- 若 $\text{dom}(f) = A$ 且 $\text{ran}(f) = B_b$，则 $A \cong B_b$
- 若 $\text{dom}(f) = A_a$ 且 $\text{ran}(f) = B$，则 $A_a \cong B$

**答案：**
任意两个良序集可比，一个必序同构于另一个的前段。

- 这由构造保证

- $f$ 保序：$a < b \Leftrightarrow f(a) \in f(b) \Leftrightarrow f(a) < f(b)$

由序数三歧性，$\alpha = \beta$

**答案：**
每个良序集序同构于唯一的序数，这称为其序型。

- $\alpha = \sup\{\beta : \beta < \alpha\}$
- 故 $\bigcup \alpha = \alpha$

### 第346题
**题目描述：**
证明：序数乘法对加法满足左分配律：$\alpha \cdot (\beta + \gamma) = \alpha \cdot \beta + \alpha \cdot \gamma$。

**解答过程：**
对 $\gamma$ 用超限归纳，固定 $\alpha$ 和 $\beta$。

**基例：** $\gamma = 0$
- $\alpha \cdot (\beta + 0) = \alpha \cdot \beta$
- $\alpha \cdot \beta + \alpha \cdot 0 = \alpha \cdot \beta + 0 = \alpha \cdot \beta$

**后继步：** 假设对 $\gamma$ 成立，证对 $\gamma + 1$
- $\alpha \cdot (\beta + (\gamma + 1)) = \alpha \cdot ((\beta + \gamma) + 1)$
- $= \alpha \cdot (\beta + \gamma) + \alpha$
- $= (\alpha \cdot \beta + \alpha \cdot \gamma) + \alpha$（归纳假设）
- $= \alpha \cdot \beta + (\alpha \cdot \gamma + \alpha)$
- $= \alpha \cdot \beta + \alpha \cdot (\gamma + 1)$

**极限步：** 设 $\lambda$ 是极限序数
- $\alpha \cdot (\beta + \lambda) = \alpha \cdot \sup\{\beta + \gamma : \gamma < \lambda\}$
- $= \sup\{\alpha \cdot (\beta + \gamma) : \gamma < \lambda\}$
- $= \sup\{\alpha \cdot \beta + \alpha \cdot \gamma : \gamma < \lambda\}$（归纳假设）
- $= \alpha \cdot \beta + \sup\{\alpha \cdot \gamma : \gamma < \lambda\}$
- $= \alpha \cdot \beta + \alpha \cdot \lambda$

**答案：**
$\alpha \cdot (\beta + \gamma) = \alpha \cdot \beta + \alpha \cdot \gamma$，左分配律成立。

计算 $\omega^\omega$ 并描述其序型。

**解答过程：**
$\omega^\omega = \sup\{\omega^n : n < \omega\}$。

**序型描述：**
- $\omega^\omega$ 是 $\omega$ 个 $\omega^n$ 的序列的上确界
- 可以看作有限序列的集合按字典序排列
- 或者 $\omega$ 层嵌套的 $\omega$ 序列

**性质：**
- $\omega^\omega$ 是极限序数
- $\omega^\omega$ 是可数的（可数序数的可数上确界可数）
- $\omega^\omega < \omega_1$

**与其他序数比较：**
- $\omega^n < \omega^\omega$ 对所有有限 $n$
- $\omega^\omega \cdot \omega > \omega^\omega$

**答案：**
$\omega^\omega$ 是可数极限序数，是所有有限幂 $\omega^n$ 的上确界。

### 第349题
**题目描述：**
证明：$\varepsilon_0 = \omega^{\varepsilon_0}$ 是最小的满足 $\omega^\alpha = \alpha$ 的序数 $\alpha > \omega$。

**解答过程：**
**定义：** $\varepsilon_0 = \sup\{\omega, \omega^\omega, \omega^{\omega^\omega}, \ldots\}$

**证明 $\omega^{\varepsilon_0} = \varepsilon_0$：**
- $\omega^{\varepsilon_0} = \omega^{\sup\{\omega_n : n < \omega\}}$（其中 $\omega_0 = \omega$，$\omega_{n+1} = \omega^{\omega_n}$）
- $= \sup\{\omega^{\omega_n} : n < \omega\}$
- $= \sup\{\omega_{n+1} : n < \omega\}$
- $= \sup\{\omega_n : n < \omega\} = \varepsilon_0$

**证明是最小的：**
设 $\alpha > \omega$ 且 $\omega^\alpha = \alpha$。
- 若 $\alpha < \varepsilon_0$
- 则 $\alpha$ 在某处被序列 $\omega_n$ 超过
- 设 $n$ 是最小使 $\alpha \leq \omega_n$
- 若 $\alpha < \omega_n$，则 $\omega^\alpha < \omega^{\omega_n} = \omega_{n+1}$
- 递推可知 $\alpha$ 不能是不动点
- 矛盾

**答案：**
$\varepsilon_0$ 是序数指数函数 $\omega^\alpha$ 的最小不动点。

### 第350题
**题目描述：**
证明：$\varepsilon_0$ 是可数序数且 $\varepsilon_0 < \omega_1$。

**解答过程：**
**证明 $\varepsilon_0$ 可数：**
- $\varepsilon_0 = \sup\{\omega_n : n < \omega\}$
- 其中 $\omega_0 = \omega$，$\omega_{n+1} = \omega^{\omega_n}$
- 每个 $\omega_n$ 可数（用归纳：$\omega$ 可数，若 $\omega_n$ 可数则 $\omega^{\omega_n}$ 可数）
- 可数多个可数序数的可数序列的上确界可数
- 因此 $\varepsilon_0$ 可数

**证明 $\varepsilon_0 < \omega_1$：**
- $\omega_1$ 是所有可数序数的集合
- $\varepsilon_0$ 是可数序数
- 故 $\varepsilon_0 \in \omega_1$，即 $\varepsilon_0 < \omega_1$

**意义：**
$\varepsilon_0$ 虽然在直观上"非常大"，但仍然是可数的，远小于第一个不可数序数 $\omega_1$。

**答案：**
$\varepsilon_0$ 是可数序数，$\varepsilon_0 < \omega_1$。
