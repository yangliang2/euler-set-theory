### 问题 051

**题目描述：**
设 $A = \{1, 2, 3, 4\}$，$B = \{3, 4, 5, 6\}$，求 $A \cup B$。

**解答过程：**
并集 $A \cup B$ 包含所有属于 $A$ 或属于 $B$ 的元素。
$A$ 的元素：$1, 2, 3, 4$
$B$ 的元素：$3, 4, 5, 6$
合并去重后：$1, 2, 3, 4, 5, 6$

**答案：**
$$A \cup B = \{1, 2, 3, 4, 5, 6\}$$

### 问题 062

**题目描述：**
证明分配律：$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$。

**解答过程：**
设 $x \in A \cap (B \cup C)$
则 $x \in A$ 且 $x \in B \cup C$
即 $x \in A$ 且 ($x \in B$ 或 $x \in C$)
这等价于 ($x \in A$ 且 $x \in B$) 或 ($x \in A$ 且 $x \in C$)
即 $x \in A \cap B$ 或 $x \in A \cap C$
所以 $x \in (A \cap B) \cup (A \cap C)$

**答案：**
分配律成立：$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$

所以 $x \in (A \setminus B) \cup (B \setminus A) = A \triangle B$

等式成立：$A \triangle B = (A \cup B) \setminus (A \cap B)$

利用特征函数：$\chi_{A \triangle B} = \chi_A + \chi_B \pmod{2}$

$(A \triangle B) \triangle C$ 的特征函数：
$\chi_{(A \triangle B) \triangle C} = \chi_{A \triangle B} + \chi_C = (\chi_A + \chi_B) + \chi_C \pmod{2}$

$A \triangle (B \triangle C)$ 的特征函数：
$\chi_{A \triangle (B \triangle C)} = \chi_A + \chi_{B \triangle C} = \chi_A + (\chi_B + \chi_C) \pmod{2}$

由于加法满足结合律，两者相等。

**答案：**
对称差满足结合律：$(A \triangle B) \triangle C = A \triangle (B \triangle C)$

### 问题 100

**题目描述：**
设 $A, B, C$ 是任意集合，证明广义分配律：$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$ 和 $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$ 的等价性。

**解答过程：**

即 $x \in A \cup B$ 且 $x \in A \cup C$
所以 $x \in (A \cup B) \cap (A \cup C)$

**答案：**
两个分配律等价且都成立，这是集合运算的基本性质。
