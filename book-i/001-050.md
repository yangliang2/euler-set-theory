### 问题 001

**题目描述：**
用列举法表示集合 $A = \{x \in \mathbb{Z} \mid -3 < x \leq 4\}$。

**解答过程：**
根据条件 $-3 < x \leq 4$ 且 $x \in \mathbb{Z}$，整数 $x$ 的取值范围为：$-2, -1, 0, 1, 2, 3, 4$。

**答案：**
$$A = \{-2, -1, 0, 1, 2, 3, 4\}$$

### 问题 002

**题目描述：**
用描述法表示集合 $B = \{1, 3, 5, 7, 9\}$。

**解答过程：**
观察集合中的元素，它们都是小于10的正奇数，即被2除余1的正整数。

**答案：**
$$B = \{x \in \mathbb{Z}^+ \mid x < 10, x \equiv 1 \pmod{2}\}$$
或
$$B = \{2n - 1 \mid n \in \mathbb{Z}, 1 \leq n \leq 5\}$$

### 问题 004

**题目描述：**
判断下列命题的真假：$\emptyset \subseteq \emptyset$

**解答过程：**
根据子集定义，$A \subseteq B$ 当且仅当对任意 $x$，若 $x \in A$ 则 $x \in B$。
对于 $A = B = \emptyset$，前提 "x \in \emptyset" 恒假，因此整个蕴含式恒真。

**答案：**
真（True）

### 问题 016

**题目描述：**
证明：$A \subseteq B$ 当且仅当 $A \cup B = B$。

**解答过程：**
**必要性** $(\Rightarrow)$：设 $A \subseteq B$。
- 显然 $B \subseteq A \cup B$。
- 对任意 $x \in A \cup B$，有 $x \in A$ 或 $x \in B$。
  - 若 $x \in A$，由 $A \subseteq B$ 得 $x \in B$。
  - 若 $x \in B$，则 $x \in B$。
- 故 $A \cup B \subseteq B$，因此 $A \cup B = B$。

**充分性** $(\Leftarrow)$：设 $A \cup B = B$。
- 对任意 $x \in A$，有 $x \in A \cup B = B$，故 $x \in B$。
- 因此 $A \subseteq B$。

**答案：**
证明完毕。

### 问题 017

**题目描述：**
证明：$A \subseteq B$ 当且仅当 $A \cap B = A$。

**解答过程：**
**必要性** $(\Rightarrow)$：设 $A \subseteq B$。
- 显然 $A \cap B \subseteq A$。
- 对任意 $x \in A$，由 $A \subseteq B$ 得 $x \in B$，故 $x \in A \cap B$。
- 因此 $A \subseteq A \cap B$，所以 $A \cap B = A$。

**充分性** $(\Leftarrow)$：设 $A \cap B = A$。
- 对任意 $x \in A = A \cap B$，有 $x \in B$。
- 因此 $A \subseteq B$。

**答案：**
证明完毕。

### 问题 019

**题目描述：**
证明德摩根律：$\complement_U (A \cup B) = \complement_U A \cap \complement_U B$。

**解答过程：**
对任意 $x$：
$$\begin{aligned}
x \in \complement_U (A \cup B) 
&\Leftrightarrow x \notin A \cup B \\
&\Leftrightarrow x \notin A \text{ 且 } x \notin B \\
&\Leftrightarrow x \in \complement_U A \text{ 且 } x \in \complement_U B \\
&\Leftrightarrow x \in \complement_U A \cap \complement_U B
\end{aligned}$$

**答案：**
证明完毕。此为德摩根律之一。

### 问题 027

这样的函数共有 $2^n$ 个（每个元素有2种选择）。

**答案：**
证明完毕。$|\mathcal{P}(A)| = 2^n$。

### 问题 030

**题目描述：**
证明：$A \times (B \cup C) = (A \times B) \cup (A \times C)$。

**解答过程：**
对任意 $(x, y)$：
$$\begin{aligned}
(x, y) \in A \times (B \cup C) 
&\Leftrightarrow x \in A \text{ 且 } y \in B \cup C \\
&\Leftrightarrow x \in A \text{ 且 } (y \in B \text{ 或 } y \in C) \\
&\Leftrightarrow (x \in A \text{ 且 } y \in B) \text{ 或 } (x \in A \text{ 且 } y \in C) \\
&\Leftrightarrow (x, y) \in A \times B \text{ 或 } (x, y) \in A \times C \\
&\Leftrightarrow (x, y) \in (A \times B) \cup (A \times C)
\end{aligned}$$

**答案：**
证明完毕。笛卡尔积对并集满足分配律。

### 问题 034

\end{aligned}$$

证明完毕。

设 $A = \{1, 2\}$，$B = \{a, b\}$，$C = \{x, y\}$，求 $(A \times B) \cap (A \times C)$。

**解答过程：**
$$\begin{aligned}
A \times B &= \{(1, a), (1, b), (2, a), (2, b)\} \\
A \times C &= \{(1, x), (1, y), (2, x), (2, y)\} \\
(A \times B) \cap (A \times C) &= \emptyset \quad \text{（因为 } B \cap C = \emptyset\text{）}
\end{aligned}$$

**答案：**
$$\emptyset$$

### 问题 048

**题目描述：**
证明：若 $A \subseteq B$ 且 $C \subseteq D$，则 $A \times C \subseteq B \times D$。

**解答过程：**
对任意 $(x, y) \in A \times C$：
- $x \in A$，由 $A \subseteq B$ 得 $x \in B$。
- $y \in C$，由 $C \subseteq D$ 得 $y \in D$。
故 $(x, y) \in B \times D$。
因此 $A \times C \subseteq B \times D$。

**答案：**
证明完毕。

### 问题 050

**题目描述：**
设 $A, B, C$ 是任意集合，证明：$A \setminus (B \cup C) = (A \setminus B) \cap (A \setminus C)$。

**解答过程：**
$$\begin{aligned}
A \setminus (B \cup C) &= A \cap \complement_U(B \cup C) \\
&= A \cap (\complement_U B \cap \complement_U C) \\
&= (A \cap \complement_U B) \cap (A \cap \complement_U C) \\
&= (A \setminus B) \cap (A \setminus C)
\end{aligned}$$

**答案：**
证明完毕。

## 卷末总结

本卷涵盖的集合论基础概念：

| 概念 | 符号 | 说明 |
|------|------|------|
| 属于关系 | $\in$ | 元素与集合的关系 |
| 子集 | $\subseteq$ | 集合与集合的包含关系 |
| 真子集 | $\subsetneq$ | 真包含关系 |
| 空集 | $\emptyset$ | 不含任何元素的集合 |
| 全集 | $U$ | 讨论范围的总集合 |
| 并集 | $\cup$ | $A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$ |
| 交集 | $\cap$ | $A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$ |
| 差集 | $\setminus$ | $A \setminus B = \{x \mid x \in A \text{ 且 } x \notin B\}$ |
| 补集 | $\complement_U A$ | 全集中不属于 $A$ 的元素 |
| 幂集 | $\mathcal{P}(A)$ | $A$ 的所有子集构成的集合 |
| 笛卡尔积 | $\times$ | $A \times B = \{(a, b) \mid a \in A, b \in B\}$ |
| 对称差 | $\oplus$ | $A \oplus B = (A \setminus B) \cup (B \setminus A)$ |

**重要性质：**
- 德摩根律：$\complement_U(A \cup B) = \complement_U A \cap \complement_U B$
- 分配律：$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- 吸收律：$A \cap (A \cup B) = A$，$A \cup (A \cap B) = A$
